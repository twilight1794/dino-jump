Dino Jump es muy similar a otros juegos que existen por ahí, incluyendo el original de Google Chrome. Pero además de sus funciones, esta implementación posee una característica que lo distingue de las demás impleentaciones.

Libertad

Dino Jump está publicado bajo la licencia GNU GPL v3, lo que es bueno para ti, ya que tienes cuatro libertades:

De ejecutar el programa a como se desee
Sin restricciones, ni ningún tipo de trabas, o funcionalidades bloqueadas. Tú eres libre de usar el programa a como quieras.
De poder estudiar cómo funciona el programa
Si te gusta programar, puedes usar el código fuente de este juego para entender como funciona, y si crees que puedes añadirle alguna funcionalidad o corregir algún error, ¡sé valiente y modifica el programa!
De poder distribuir copias del programa
Si quieres descargar el juego, y compartirlo con tus amigos, eres libre de hacerlo, de hecho, te instamos a que lo hagas.
De poder distribuir tus modificaciones del programa
Si llegases a crear una nueva versión derivada de este juego, eres libre de distribuir tu versión a como quieras, siempre y cuando tu versión siga siendo libre.
